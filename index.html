Data Privacy

This app does not permanently process any personal data. Player names entered by users are stored only locally and temporarily in the working memory (RAM) of the device in use. No permanent storage, transmission, or further processing of this data takes place.

The provider has no access to this data at any time – neither technically nor legally. All entries remain solely on the user's device.

Purpose of Data Processing

The entry of a player name serves exclusively to identify the participants during an ongoing game session.

Legal Basis

Processing is based on Art. 6 (1) lit. b GDPR (contract performance).

Storage Duration

The entered data (player names) are stored only for the duration of the current game session in RAM and are completely deleted when the app is closed or restarted. No permanent storage takes place.

Notice

Entering a player name is voluntary but required to use the app's game functions. The use of a pseudonym is possible at any time.
